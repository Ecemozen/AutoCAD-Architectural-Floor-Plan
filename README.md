# AutoCAD Architectural Floor Plan and Interior Layout Design

This repository contains a 2D architectural floor plan, interior layout, and technical drawing project developed using Autodesk AutoCAD. The project systematically illustrates the sequential phases of architectural drafting, transitioning from raw geometric partitions to a fully dimensioned, annotated, and structured layout ready for industrial deployment.

The structural workflow strictly adheres to international drawing standards, emphasizing precise layer management, procedural hatch applications, optimized block organization, and professional annotation methodologies.

## Technical Specifications and Engineering Standards

* **Core Platform:** Autodesk AutoCAD
* **Project Type:** 2D Architectural Floor Plan and Space Planning
* **Spatial Scale:** 22.75 m² (Main Living Space / Salon)
* **Sequential Production Pipeline:**
  1. **Phase 1 (Structural Layout):** Construction of primary geometric boundaries, external and internal wall partitions, and precise calibration of door and window structural openings.
  2. **Phase 2 (Interior Layout & Furnishing):** Implementation of functional interior zoning utilizing standardized blocks for furniture and architectural assets to evaluate spatial circulation.
  3. **Phase 3 (Material Hashing & Texturing):** Application of procedural patterns (hatch) to differentiate floor materials, structural elements, and architectural textures.
  4. **Phase 4 (Annotation, Dimensioning & Title Block):** Execution of comprehensive external linear dimensioning, exact spatial area calculations, and integration into a professional ISO-compliant A1 layout title block format.

## Repository Structure

* `EcemÖzenDrawing1-ISO A1 Layout.pdf`: Vectorized output document displaying the four primary evolution stages, lineweights, and title block presentation without scale distortion.
* `EcemÖzenDrawing1.dwg`: Original AutoCAD source asset containing complete geometric datasets, layered structures, and reusable block references.
